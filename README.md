## 구현된 기능

### 7개 Amazon A+ 모듈
1. **Module 1 — Hero Banner**: 메인 비주얼, 헤드라인, USP 배지
2. **Module 2 — Key Benefits**: 핵심 장점 4카드 그리드
3. **Module 3 — Flavor Lineup**: 4가지 맛 인터랙티브 카드
4. **Module 4 — Texture & Experience**: 식감 강조 다크 섹션
5. **Module 5 — Convenience & Packaging**: 편의성 3카드 + 패키지 배너
6. **Module 6 — Brand Story**: 한국 디저트 문화 스토리 섹션
7. **Module 7 — Comparison Chart**: 맛별 비교표 + 플레이버 필터

### 인터랙션 기능
- 스티키 네비게이션 (스크롤 시 표시)
- 플레이버 필터 버튼 (비교표 컬럼 토글)
- 스크롤 페이드업 애니메이션
- 레이팅 바 스크롤 트리거 애니메이션
- 플레이버 카드 3D 패럴랙스 호버 효과
- 브랜드 스토리 이미지 스택 리빌
- 히어로 이미지 float 애니메이션
